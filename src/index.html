Vue Hotkeys

The Vue Hotkeys component helps you to smoothly implement keyboard shortcuts to any Vue application.

Try the following hotkeys:

  • ⌘ + S
  • ⌘ + D
  • ⌘ + ⬅️
  • ⌘ + ➡️

Note: For using Arrow Keys you have to pass below Strings to the shortcuts prop

Arrow String (prop) to be passed Code
Arrow Left ⬅️ ArrowLeft 37
Arrow Up ⬆️ ArrowUp 38
Arrow Right ➡️ ArrowRight 39
Arrow Down ⬇️ ArrowDown 40
